# 🖥️ Legacy AI Assistant (Windows XP & 7)

A lightweight, modern AI assistant client beautifully optimized for legacy operating systems. Bring the power of modern Artificial Intelligence straight to your vintage setup without bloated requirements.

---

## ✨ Features

* **True Retro Compatibility:** Runs natively on **Windows XP** and **Windows 7**.
* **Zero System Patches Required:** Works out-of-the-box. No need to install complex TLS 1.2 patches or framework updates on your legacy machine.
* **Lightweight & Fast:** Low RAM and CPU footprint, designed to respect the performance of classic hardware.
* **OpenRouter Powered:** Connects to modern AI models through a flexible API key.

---

## 🔑 Prerequisites: Getting Your OpenRouter API Key

This application uses **OpenRouter** to connect to the internet and talk to the AI. Before launching the app, you need to get a free API key:

1. Go to the official website: [OpenRouter.ai](https://openrouter.ai/)
2. Sign in or create a free account.
3. Navigate to your dashboard and go to **Keys** (or API Keys).
4. Click **Create Key**, give it a name (e.g., *XP AI App*), and copy the generated key.
   > ⚠️ **Important:** Copy your key immediately and save it somewhere safe. You will need to paste it into the application upon its first launch.

---

## 🚀 Installation & Setup Guide

### Step 1: Download the Installer
Go to the **Files** tab here on SourceForge and download the latest setup executable (e.g., `xp_ai_setup.exe`).

### Step 2: Run the Setup
1. Double-click the downloaded setup file.
2. Follow the classic installation wizard instructions.
3. Choose your installation path (by default, it installs into `Program Files`).
4. Check the box to **Create a desktop shortcut** to get that sweet, custom CRT Monitor icon on your desktop.

### Step 3: First Run & Activation
1. Launch the application using the desktop icon or the Start Menu shortcut.
2. On the first screen, the application will prompt you for your **OpenRouter API Key**.
3. Paste the key you generated in the prerequisites step and click **Save/Connect**.
4. That's it! You are now ready to chat with AI directly from your vintage rig.
